# Crup App
Modern web development involves user interaction with databases. As a rule, the user needs to perform some actions with the database.

These actions are 4:
- create something 
- display 
- update
- delete

<ul>

 	<li>JAVASCRIPT: Used to create interactive web pages.</li>
</ul>
